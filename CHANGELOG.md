# Changelog

All notable changes to this package will be documented here.

## [0.1.0] - 2025-12-03

- Added: `PrimaryButton`, `AppTextField` variants (`Outlined`, `Filled`, `Underline`), `AppSnackbar`, `AppDialog`, `formatters`, `loggers`, and `validators`.
- Migration: Converted several widgets to use `flutter_hooks` and added the `flutter_hooks` dependency to `pubspec.yaml`.
